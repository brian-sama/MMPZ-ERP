# Unified Enterprise Portal Frontend

React + Vite frontend for the ERP + Intranet unified dashboard.

## Start

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Structure

```text
src
├ app
│  ├ dashboard
│  ├ erp
│  ├ intranet
│  └ administration
├ components
│  ├ ui
│  ├ layout
│  └ widgets
├ services
├ hooks
└ utils
```
